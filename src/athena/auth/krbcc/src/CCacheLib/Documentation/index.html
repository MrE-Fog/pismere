CCacheLib
What is CCacheLib?

CCacheLib is a CFM shared library for use with MIT Kerberos for Macintosh. CCacheLib provides an API for accessing the in-memory credentials cache for Kerberos v4 and Kerberos v5 tickets. CCacheLib is used by other libraries in MIT Kerberos Library and should not normally be needed by Kerberos applications.

Where is the API specification?

There are two versions of the CCacheLib API available to callers. The most recent API is CCacheLib API v3, while CCacheLib API v2 is available for compatibility.

Where do I get CCacheLib?
The CCacheLib SDK is part of MIT Kerberos for Macintosh. You can get the latest release of the MIT Kerberos for Macintosh here.

Please read the CCacheLib Bugs List and Version History for information on the latest release of CCacheLib.
